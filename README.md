\# Zara Sales - Exploratory Data Analysis



\## Overview

This project performs Exploratory Data Analysis (EDA) on Zara's sales data to uncover insights about pricing, promotions, seasons, and product categories.



\## Tools \& Libraries Used

\- Python

\- Pandas

\- Matplotlib

\- Seaborn

\- Jupyter Notebook



\## Dataset

The dataset contains Zara product information including price, sales volume, product category, season, promotions, and product position.



\## Key Findings

\- Promotions positively impact sales volume

\- Clothing is the dominant product category

\- Product prices are concentrated in a moderate range with a few premium outliers

\- Seasonal trends influence product pricing and inventory

\- Only a limited number of products achieve extremely high sales performance



\## Project Structure

\- `Zara\_EDA.ipynb` — Main analysis notebook

\- `Zara\_sales\_EDA.csv` — Dataset used for analysis
